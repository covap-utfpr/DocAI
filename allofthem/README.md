# EXECUÇÃO
### 1 ) Insira as imagens no diretório **input/perspectiva_input**

obs: as notas podem ter as seguintes extenções: 
".jpg", ".jpeg", ".png", ".bmp", ".tif", ".tiff"

### 2 ) Execute o seguinte comando para rodar todo o pipeline

    make all
### 3 ) para executar partes do pipeline, utilize os seguintes comandos

    make run_perspectiva
<p>para executar perspectiva<p>
  
    make run_mosaico_cs
<p>para executar mosaico</p>

    make run_trata
<p>para executar trata</p>

    make run_ocr
  <p>para executar todos os ocr</p>

  obs: Os diretórios de entrada e saida devem ser ajustados dentro do Makefile.
